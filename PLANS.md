# Execution Plan: Offline “Production-Style” GaN DFT + MLIP Showcase

This plan turns the existing GaN DFT+MLIP pipeline into a recruiter-grade, reproducible, local-only demo project.
It is intentionally “demo-first”: the Streamlit UI and RAG reports must run offline using precomputed artifacts by default.

## Guiding Constraints (Non-Negotiable)
- Offline only (privacy). No external API calls.
- Streamlit demo uses precomputed artifacts by default; no DFT in the app.
- Optional MLIP relaxation allowed if GPU exists; must degrade gracefully to “view-only” mode.
- RAG scope is narrow: extract typical GaN DFT parameters from local papers, tabulate with citations, compare to local configs + local run outputs.

- Any local compute in the app must be opt-in and bounded.
